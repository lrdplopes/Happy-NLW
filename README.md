## 📁 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/lrdplopes/Happy-NLW

    #Acessar o repositório
    $ cd Happy-NLW

    #Instalar suas dependências
    $yarn install

    #Iniciar o projeto
    $yarn start

```
